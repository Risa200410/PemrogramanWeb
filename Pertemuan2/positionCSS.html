position: static

Position static digunakan untuk mengatur element menjadi statis secara deafult. Elemen akan mengikuti posisi normal secara default, elmeen tidak dipengaruhi oleh properti seperti top, bottom, left, dan right.:

Elemen ini diatur mengggunakan posisi: static;




position:relative;

Sebuah element HTML yang menggunakan position relative akan terletak pada posisi normal. Mengatur properti atas, kanan, bawa, dan kiri dari elemen yang diposisilan dengan "relatif" akan membuatnya jauh dari posisi normalnya. Konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.:

Elemen ini diatur menggunakan posisi : relative;



position: fixed;

Sebuah element HTML yang di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll. Berlaku pengaturan left, bottom, top, dan right pada position fixed:

Elemen ini diatur menggunakan posisi: fixed;















position: absolute;

Element HTML yang menggunakan position absolute akan diposisikan relatif dengan elemen lain yang mendahuluinya yang terdekat. bukan relatif terhadap layar secra normal

Elemen ini diatur menggunakan posisi: relative;
Elemen ini diatur menggunakan posisi: absolute
